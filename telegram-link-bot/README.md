# telegram-link-bot

`npm ci` → `node server.mjs`

Env: `TELEGRAM_BOT_TOKEN`, `TELEGRAM_LINK_SITE_URL`, `TELEGRAM_LINK_INTERNAL_API_TOKEN`, `TELEGRAM_BOT_INCOMING_SECRET`; при необходимости `TELEGRAM_WEBHOOK_BASE_URL` (HTTPS, иначе long polling), `PORT`, `TELEGRAM_SUPPORT_GROUP_ID`, `TELEGRAM_ADMIN_TELEGRAM_IDS`.
